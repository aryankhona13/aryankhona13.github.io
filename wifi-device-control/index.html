⚙️ System Architecture & Hardware Specs

This project utilizes a high-performance, dual-microcontroller architecture to ensure no network bottlenecks occur during complex hardware operations.

🧠 ESP8266 (Gateway)

Hosts a local Web-Server in AP/STA mode. Receives HTTP commands and transmits serial data to the main processor.

⚡ Arduino Nano (Core)

The main workhorse. Executes Non-Blocking logic for precision relay timing and Trigonometric NeoPixel Math.

🔌 Relays & Indicators

3-Channel active-HIGH BC547 controlled relays for AC appliances, backed by physical LED status indicators.

🌈 WS2812B NeoPixel

An 8-LED intelligent ring displaying 9 unique mathematical sub-pixel animations and system statuses.

📡 Wireless Connection Guide

🟢 Access Point (AP) Mode

Use this for standalone operation without a home router.

  1. Power on the WiControl hardware.
  2. Open your phone's Wi-Fi settings.
  3. Connect to the network: "WiControl".
  4. Open the downloaded App and start controlling!

🏠 Station (STA) Mode

Use this to integrate the hardware into your existing home network.

  1. Update the ESP8266 code with your Home Wi-Fi SSID & Password.
  2. Connect your phone to your Home Wi-Fi.
  3. Enter the ESP's allocated IP address in the WiControl App settings.

📱 Application Features & Modes

The WiControl app provides a modular interface to control physical appliances with millisecond precision:

🎛️ Manual Dashboard

Instant ON/OFF control for individual relays (R1, R2, R3) and an emergency "ALL ON/ALL OFF" master override switch.

🏃 Dynamic Chaser

Creates running light effects. Choose from 4 sequences (Forward, Reverse, Ping-Pong, Alternate) and define exact millisecond speeds.

⏱️ Smart Timer

Set a countdown duration (1 Min, 5 Min, 10 Min, 30 Min, 1 Hour). The selected relay will activate and auto-shutoff when the time expires.

🔄 Infinite Cyclic Mode

Define independent "ON Time" and "OFF Time" parameters. The appliance will continuously loop between states automatically.

🚥 Custom Sequence

Program a 3-step timeline. Set custom active delays for Relay 1, 2, and 3. They will trigger sequentially based on your input.

🎉 Party Interlock

A fast-paced algorithm that triggers random states across all 3 relays to create a disco-strobe lighting effect.

🌈 The Ultra-Pro NeoPixel Engine

The system features a highly advanced, non-blocking NeoPixel engine. Every effect utilizes the full 65,000 HSV color spectrum.

  • 1. Rainbow Vortex: A smooth, spinning multi-color classic loop.
  • 2. Rainbow Breathe: Elegant fading in and out across shifting color hues.
  • 3. Trinity Chase: Three distinct dots chasing each other with fading light trails.
  • 4. Color Wave: A sine-wave brightness ripple across a shifting rainbow.
  • 5. Rainbow Comet: A bright shooting star with a trailing tail, changing colors every cycle.
  • 6. Rainbow Radar: A ping-pong sweeping scan that continuously morphs colors.
  • 7. Color Heartbeat: A realistic double-pulse "dhak-dhak" effect in dynamic hues.
  • 8. Dual Rainbow Helix: Two glowing dots spinning in opposite directions, crossing perfectly.
  • 9. Neon Juggle: Three individual dots bouncing at distinct mathematical speeds.
  • 10. Auto-Loop: Automatically cycles through all 9 effects sequentially.
