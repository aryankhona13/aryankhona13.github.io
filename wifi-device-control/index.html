⚙️ Technical Specifications & Logic Flow

This project utilizes a high-performance, Dual-Microcontroller Architecture to prevent network bottlenecks during complex operations:

🧠 ESP8266 (Wi-Fi Gateway)

Hosts a robust local Web-Server. It receives HTTP REST commands from the Android App and relays them as clean Serial Strings to the main processor.

⚡ Arduino Nano (Core Processor)

The main workhorse. Uses a custom 'Smart Number Extractor' algorithm to parse incoming serial commands. It executes Non-Blocking millis() logic for precision relay timing.

🔌 Universal Relays (AC/DC)

3-Channel active-HIGH BC547 controlled relays. Acts as an isolated dry-contact switch, safely controlling both 220V AC and 12V/24V DC loads.

🌈 WS2812B NeoPixel Engine

An 8-LED ring driven by Trigonometric Math (Sine/Cosine waves). Utilizes a 16-bit Hue spectrum (65,535 steps) for ultra-smooth, liquid-like color morphing.

📡 Wireless Connection Guide

🟢 Access Point (AP) Mode

Use this for standalone operation without a home router.

  1. Power on the WiControl hardware.
  2. Open your phone's Wi-Fi settings.
  3. Connect to the network: "WiControl".
  4. Open the downloaded App and start controlling!

🏠 Station (STA) Mode

Integrate the hardware into your existing home network.

  1. Update the ESP8266 code with your Home Wi-Fi SSID & Password.
  2. Connect your phone to your Home Wi-Fi.
  3. Enter the ESP's IP address in the WiControl App settings.

📱 Application Features & Modes

🎛️ Manual Dashboard

Instant ON/OFF control for individual relays and an emergency "ALL ON/OFF" master override switch.

🏃 Dynamic Chaser

Creates running light effects. Choose sequences (Forward, Reverse, Ping-Pong) and define exact millisecond speeds.

⏱️ Smart Timer

Set a countdown duration. The selected relay will activate and auto-shutoff when the time expires.

🔄 Infinite Cyclic Mode

Define independent "ON Time" and "OFF Time" parameters. The appliance will loop between states automatically.

🚥 Custom Sequence

Program a 3-step timeline. Set custom active delays for Relay 1, 2, and 3 to trigger sequentially.

🎉 Party Interlock

A fast-paced algorithm triggering random states across all 3 relays for a disco-strobe effect.

🌈 The Ultra-Pro NeoPixel Engine

A proprietary Non-Blocking LED engine. Instead of standard 8-bit RGB, every effect utilizes the full 65,000 HSV color spectrum for flawless transitions.

  • 1. Rainbow Vortex: A smooth, spinning multi-color classic loop.
  • 2. Rainbow Breathe: Elegant fading in and out across shifting color hues.
  • 3. Trinity Chase: Three distinct dots chasing each other with fading light trails.
  • 4. Color Wave: A sine-wave brightness ripple across a shifting rainbow.
  • 5. Rainbow Comet: A bright shooting star with a trailing tail, changing colors every cycle.
  • 6. Rainbow Radar: A ping-pong sweeping scan that continuously morphs colors.
  • 7. Color Heartbeat: A realistic double-pulse "dhak-dhak" effect in dynamic hues.
  • 8. Dual Rainbow Helix: Two glowing dots spinning in opposite directions, crossing perfectly.
  • 9. Neon Juggle: Three individual dots bouncing at distinct mathematical speeds.
  • 10. Auto-Loop: Automatically cycles through all 9 effects sequentially.
⚠️ Technical Safety Note: The relays act as isolated switches capable of handling High-Voltage AC current. Do not modify the relay terminals while the system is connected to mains power.
